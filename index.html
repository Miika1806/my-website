I made my own website, you're looking at it.

I made my first blogpost to describe how my website functions, you can read it here.

To give a short summary I tried having a very minimal website which loads quickly and uses no JavaScript or other unnecessary features. I am very familiar with Django Web Framework and have done other websites with it (such as this text based game using MySQL), but as a personal site I prefer a basic design. This website simply runs on nginx and serves static content for users. I write my blog posts using a helpful script to automatically create static HTML pages as well as an RSS feed.

It was especially important for me to have an RSS feed as I wish more websites still offered that option.
